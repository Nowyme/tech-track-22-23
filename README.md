## Getting started 
### Installation

### Clone repository
```
clone Nowyme/tech-track-22-23
```


### Install dependencies
To install all the dependencies, type this in your terminal:

```
npm install
```


### Start the server in local
To get the application running, type this in your terminal:

```
npm run dev
```
